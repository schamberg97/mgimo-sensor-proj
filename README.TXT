В случае возникновения проблем:
	1) Не обнаружен сенсор BME-280. 
		Попробуйте запустить приложение с параметром 0x76, например "node index 0x76" (без кавычек). Если это не поможет, веротяно сенсор неправильно подключен
	2) Не работает приложение, MongoDB автоматически не установилась
		Попробуйте установить MongoDB вручную с сайта < https://docs.mongodb.com/manual/administration/install-community/ >
	3) Не работает приложение, ошибки с БД. 
		Может возникнуть на платформах, отличных от MacOS. Убедитесь, что MongoDB версии не ниже 3.6 установлен
	4) Прочие ошибки
		Попробуйте запустить приложение с параметром --debug, например "node index --debug" (без кавычек). Интерпретатор Node.JS и Приложение будет выдавать больше сообщений об ошибках
